2. Opening an Existing AADL Model

An existing AADL model can be opened in the graphical editor. Changes made to either the AADL source or the diagram will be reflected in the other.

  1. Open an existing AADL source file.

  2. Right click the AADL package or component in the Outline view.

  3. Select Open Package Diagram to open the package diagram for the package that contains the element. Select Open Classifier Diagram to open the diagram for the selected classifier. The diagram will be created if one does not already exist. The appropriate diagram will appear in the graphical editor.
