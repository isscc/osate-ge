6. Flows

The graphical editor supports creating and editing flow specifications, flow implementations, and end-to-end flows.

6.1. Creating Flow Specifications

Flow paths, flow sinks, and flow sources can be created using the palette when editing a component type.

6.1.1. Creating Flow Paths

  1. Select Flow Path from the palette.

  2. Select the feature to use as the start of the flow path.

  3. Select the feature to use as the end of the path. The new flow path will appear.

6.1.2. Creating Flow Sinks

  1. Select Flow Sink from the palette.

  2. Select the feature to use as the sink. The new flow sink will appear.

6.1.3. Creating Flow Sources

  1. Select Flow Source from the palette.

  2. Select the feature to use as the source. The new flow source will appear.

6.2. Creating Flow Implementations and End-To-End Flows

When editing a component implementation, flow implementations and end-to-end flows can be created using the Create Flow Implementation and Create End-To-End Flow buttons on the toolbar and selecting valid elements on the diagram.

6.2.1. Creating Flow Implementations

Select Create Flow Implementation from the toolbar. The Create Flow Implementation dialog will appear. The Undo button will remove the flow elements in the order they were selected.

6.2.1.1. Creating Source Flow Implementations

If valid, the flow implementation's flow specification will highlight dark orange, and flow segments will highlight purple when selected.

  1. Select a valid end source flow specification.

  2. Select a valid start element that will lead to the end source flow specification.

  3. Select the remaining elements in order leading to the end source flow specification.

  4. Click OK when done.

6.2.1.2. Creating Sink Flow Implementations

  1. Select a valid start sink flow specification on the diagram.

  2. Select a connection leading from the start sink flow specification to the next desired flow segment.

  3. Select the remaining flow segments in order leading to the end flow segment.

  4. Click OK when done.

6.2.1.3. Creating Path Flow Implementations

  1. Select a valid start path flow specification on the diagram.

  2. Select a valid connection leading from the start path flow specification's in flow end.

  3. Select the remaining elements in order leading to the start path flow specification's out flow end.

  4. Click OK when done.

6.3. Editing Flow Implementations and End-to-End Flows

When editing a component implementation, flow implementations and end-to-end flow can be edited using the flow editor. It can be accessed by right-clicking in the diagram and selecting Edit Flows... from the context menu.

6.4. Highlighting Flow Implementations and End-to-End Flows

Selecting a flow implementation or end-to-end flow from the flow drop-down in the toolbar will highlight the flow.
