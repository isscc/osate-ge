6. Flows

The graphical editor supports creating and editing flow specifications, flow implementations, and end-to-end flows.

6.1. Creating Flow Sources

Flow sources can be created when editing a component type. Create the flow source by right-clicking on an appropriate feature and selecting Create Source from the context menu. The new flow source will appear.

6.2. Creating Flow Sinks

Flow sources can be created when editing a component type. Create the flow source by right-clicking on an appropriate feature and selecting Create Sink from the context menu. The new flow sink will appear.

6.3. Creating Flow Paths

When editing a component type, flow paths can be created using the palette.

  1. Select Flow Path from the palette.

  2. Select the feature to use as the start of the flow path.

  3. Select the feature to use as the end of the path. The new flow path will appear.

6.4. Editing Flow Implementations and End-to-End Flows

When editing a component implementation, flow implementations and end-to-end flow can be edited using the flow editor. It can be accessed by right-clicking in the diagram and selecting Edit Flows... from the context menu.

6.5. Highlighting Flow Implementations and End-to-End Flows

Selecting a flow implementation or end-to-end flow from the flow drop-down in the toolbar will highlight the flow.
